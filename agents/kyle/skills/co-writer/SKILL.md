# Co-Writer Skill

Write X (Twitter) content in Josh's voice.

## Overview

This skill helps draft tweets, threads, and replies using Josh's authentic voice — the "rich uncle" Wall Street veteran who shares real experience without flexing.

## Usage

### Draft a Tweet
```
Write a tweet about [topic] in Josh's voice
```

### Draft a Reply
```
Draft a reply to this tweet: [paste tweet]
Add value using Josh's Wall Street experience
```

### Draft a Thread
```
Write a thread about [topic]
Use the lesson thread format
Include a personal story element
```

## Voice Reference

Before writing, load these files for context:
- `content/voice-profile.md` — Josh's voice characteristics
- `content/audience.md` — Who we're writing for
- `content/hooks.md` — Attention-grabbing patterns
- `content/templates/tweet-formats.md` — Structural templates
- `content/templates/reply-playbook.md` — Reply strategy

## Process

1. **Understand the ask** — What type of content? What's the topic/context?
2. **Load voice profile** — Read voice-profile.md to calibrate
3. **Check templates** — Find the right format for this content type
4. **Draft** — Write in Josh's voice using the patterns
5. **Review** — Check against forbidden words/phrases
6. **Present options** — Offer 2-3 variations when appropriate

## Content Types

| Type | Template File | Notes |
|------|---------------|-------|
| Single tweet | tweet-formats.md | Keep punchy, one idea |
| Thread | tweet-formats.md | 5-12 tweets, story-driven |
| Reply | reply-playbook.md | Add value, use experience |
| Quote tweet | tweet-formats.md | Hot take + context |

## Voice Checklist

Before finalizing content, verify:

- [ ] Sounds like Josh, not generic fintwit
- [ ] No forbidden words (leverage, optimize, utilize, etc.)
- [ ] Uses contractions naturally
- [ ] Has specific details (not vague claims)
- [ ] Passes the "would I stop scrolling?" test
- [ ] Appropriate length for content type

## Examples

### Good Output
```
Market's down 3% and everyone's asking if this is the crash.

I've traded through 6 "crashes" in 15 years. Most of them weren't.

The pattern is always the same:
- Fast drop
- Panic selling
- Slow recovery
- Everyone pretends they bought the dip

The people selling today bought last month. Don't be that guy.
```

### Bad Output
```
The market is experiencing volatility today. It's important to remain calm during periods of uncertainty. Consider this an opportunity to evaluate your portfolio and potentially add to positions at lower prices.
```
(Too generic, no personality, hedge words)

## Integration

The co-writer skill is meant to:
1. Speed up content creation
2. Maintain voice consistency
3. Overcome writer's block
4. Draft starting points Josh can edit

Josh always reviews and edits before posting. This is a drafting tool, not an autopilot.
